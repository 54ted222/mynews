# 歡迎來到我的新聞頁

這是一個用 **React + Vite + shadcn/ui** 打造的個人新聞頁，部署在 GitHub Pages 上。

## 特色

- 新聞內容以 Markdown 撰寫，放在 `src/news/` 資料夾
- 自動從檔名解析日期，並依日期排序
- 支援 frontmatter 標題、摘要與標籤
- 使用 shadcn/ui 元件，支援深色模式

## 如何新增文章

在 `src/news/` 底下新增一個 `.md` 檔即可，格式如下：

```md
---
title: 標題
date: 2026-04-21
tags: 標籤1, 標籤2
summary: 摘要
---

正文內容……
```
