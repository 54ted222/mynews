[^super-app]: Microsoft Copilot Super App，Fortune 5/29 獨家披露的內部代號「Delivering one Copilot」整合計畫，由 5 月新任 Copilot 負責人 Jacob Andreou 操盤，把 GitHub Copilot、Copilot Chat、Cowork、Autopilot 四條原本分散的產品線收進單一桌面 app，並保留 super app 與各 Copilot 之間的切換 toggle，預定 2026 暑假末（end of summer）上線。

[^cowork]: Copilot Cowork，Microsoft Copilot 家族中專責「多人即時協作」的子產品，定位類似 Google Docs 的協作層但嵌入 Microsoft 365 與 Teams 工作流，是 Super App 計畫中要收編的四大原生模組之一，與 coding（GitHub Copilot）、chat、agentic（Autopilot）並列。

[^autopilot]: Copilot Autopilot mode，CLI / IDE 內的全自動 agentic 模式，特性是自動批准 tool call、自動 retry、自動回應 tools 的提問避免 stall，無須使用者逐步點選，是 Build 2026 BRK207 session「GitHub Copilot in Visual Studio: Agents That Debug, Profile, Test」與 `/fleet` 並列的兩大主軸。

[^war]: Windows Agent Runtime（WAR），Microsoft 在 Build 2026 預計揭曉的系統級 agent runtime，以背景 service 形式管理 agent 的 lifecycle、memory 與權限，建構在 WinRT 之上加入 rule engine 做細粒度存取控制，使 agent 可直接呼叫 file manager、task scheduler 與硬體感測器，被視為 OS 級 agent system primitive。

[^bumblebee]: Bumblebee，Perplexity 於 2026-05-22 以 Apache 2.0 授權開源的單檔 Go binary 供應鏈掃描器，專攻本地 dev endpoint，可掃 npm / pnpm / Yarn / Bun / PyPI / Go modules / RubyGems / Composer 套件，以及 MCP server config、IDE extension、browser extension 與 lockfile / package cache；設計上 read-only、不執行 install script，避免掃描動作本身觸發 supply chain worm。

[^chatgphish]: ChatGPhish，Permiso Security 於 2026-05-29 揭露的 ChatGPT 漏洞，攻擊面為「ChatGPT 對剛 summarize 的第三方頁面內 markdown link 與 image URL 自動信任、自動 fetch」，可被植入偽造系統警示、把惡意連結 surface 成 assistant 親回的可點 element，並以 S3 出 QR code 繞過 desktop URL filter；Permiso 4/29 經 Bugcrowd 通報 OpenAI，被標 not reproducible 後再標 duplicate，5/29 公開時尚未確認修復。
