[^super-app]: Copilot Super App 指 Microsoft 規劃將 GitHub Copilot、Chat、Cowork、Autopilot 統合進單一行動／桌面 app 的產品計畫，由 Jacob Andreou 操盤、暑末（8 月底）launch。定位類似亞洲市場 WeChat 級「一個入口包多服務」的概念，差別在於入口是開發者工作流而非消費端。

[^mcp]: Model Context Protocol，Anthropic 2024 年底主推的開放協定，讓 LLM 客戶端以統一介面呼叫外部工具（API、檔案、資料庫）。透過 MCP server 揭露 scoped tool、用 OAuth 2.1 控管權限。Gemini Spark、Claude Desktop、Cursor 等已陸續支援，正在成為 agent 與 SaaS 對接的事實標準。

[^mythos]: Anthropic 開發中的 AI 安全研究模型，主打自動發現程式碼漏洞與生成 exploit。Project Glasswing 內測掃 1,000+ OSS 專案、回報 23,019 個潛在漏洞、6 家獨立資安公司驗證 90.6% true positive。2026-05-28 隨 Series H 公告「coming weeks」公開，但定價、入口、模型權重是否開放尚未揭。

[^war]: Windows Agent Runtime，Microsoft 預期在 Build 2026 揭曉的 OS 級 agent 基礎元件。以背景 service 形式管理 agent lifecycle、memory、權限，建在 WinRT 之上加 rule engine 做細粒度存取控制，讓 agent 可直接呼叫檔案管理、排程器、硬體感測器，不再被困在 Electron／Tauri 殼裡。

[^antigravity]: Google 自家的 agentic harness（執行框架），驅動 Gemini Spark 等 24/7 雲端 agent 在 Google 伺服器上長時間執行任務。負責規劃、工具呼叫、錯誤處理、狀態保存，使 agent 在使用者裝置關機後仍可持續監控 inbox、執行排程 workflow。

[^pru]: Premium Request Unit，GitHub Copilot 在 2025–2026 年初使用的舊計費單位，按「一次互動」計一個 PRU、用完依方案限速或拒絕。6/1 起被 AI Credits（token 計價）取代，社群實測同樣的 4 月用量在新制下從 $39 PRU 暴增到 $902 token billing，是引發 backlash 的關鍵錨點。
