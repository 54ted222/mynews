[^polymarket]: Polymarket 是建在 Polygon 上的 crypto prediction market 平台，使用者用 USDC 對未來事件結果押注、即時撮合形成的市場價格被視為「群眾資金加權」的機率估計；爭議事件結算交由 UMA optimistic oracle 仲裁。對 indie 而言，動態 odds 比個人主觀預期更可被外部稽核，常被引用為 SLO 文件裡的 dataset。

[^workflowagent]: WorkflowAgent 是 Vercel AI SDK 7 的 durable execution primitive，把 agent 的工具呼叫、reasoning step、外部 API 結果序列化到持久化儲存，可跨 process 重啟、跨部署 resume，等同把 agent loop 變成可斷點續傳的工作流；對比一般 in-memory agent，崩潰或 timeout 後不必重跑整個對話。

[^mcp-apps]: MCP Apps 是 MCP 2026-07-28 release candidate 透過 SEP-1865 提案引入的擴充，允許 MCP server 回傳 server-rendered UI 片段，host 端以沙箱化 iframe 載入並透過 postMessage 與 agent 雙向溝通；目標是讓「產品內 UI」可以被當作 tool 嵌進 agent 對話，PLG 渠道從 web app 擴張到 LLM client。

[^colorado-ai-act]: Colorado AI Act 全稱 SB 24-205 Consumer Protections for Artificial Intelligence Act，由 Colorado 州議會 2024 年通過、2026-06-30 對「高風險 AI 系統」的 deployer 與 developer 生效，涵蓋僱用、教育、金融、住房、法律、政府、醫療七類；要求風險管理計畫、年度影響評估、對 consumer 揭露 AI 介入並提供 adverse decision 救濟，州 Attorney General 主管執法。

[^effort-pricing]: Effort-Based Pricing 是 Replit 2026-07-02 全量切換的計費模式，將原先「每次 agent edit checkpoint 計一次費」改為依 agent 實際消耗的 reasoning / tool effort 計算；對 indie 的影響是月帳波動加大，官方建議用 per-user spend cap 搭配 monthly aggregate cap 雙保險避免單一 session 燒爆預算。

[^cordyceps]: Cordyceps 是 Novee Security 於 2026 年 6 月揭露的 GitHub Actions 攻擊鏈代稱，利用 `pull_request_target` 在 fork PR 觸發時繼承高權限 secrets 的設計缺陷，搭配 `actions/checkout` 未鎖 ref 的舊版默認行為，從 untrusted PR 注入惡意 workflow 偷 token；GitHub 在 v7 改了默認、要求 v3–v6 在 2026-07-16 前 backport 修補。
