[^anthropic-kyc]: Anthropic 自 2026-07-08 起對 Claude Free / Pro / Max 消費 tier 帳號啟用的身分驗證機制，由 Persona 服務端保管臉部幾何生物特徵；scope 限消費 tier、Team / Enterprise / Platform 依 Commercial Terms 豁免，Claude API 也完全豁免（以 API key 驗證、不進消費 tier 帳號系統）；觸發是「small subset flagged accounts + appeal route」而非全體強制。中國國家身分證不被接受，無護照的中國用戶會被鎖在門外。

[^fable-5]: Claude Fable 5 為 Anthropic 現階段最頂級的 GA 模型，定價 `$10 / $50 per million tokens`（input / output），是 Opus 4.8 兩倍、Sonnet 5 促銷價五倍。Pro / Max / Team 週限保護窗自 2026-07-07 起改為 usage credits 制，需預先啟用 credits pool；未啟用時 pool 一燒完即 mid-session 直接中斷、無自動退回 metered billing 的 fallback，等於 agent loop 半途死機。

[^cordyceps]: Cordyceps 指 GitHub Actions 於 2026-06-18 公告的 `actions/checkout` 安全 baseline 收緊——將 `pull_request_target` 事件的 checkout 預設從「fork PR 的 head SHA」改為「base branch」，避免 fork code 在特權 context 執行。原本鎖在 v7 major 的變更會於 2026-07-16 backport 到 v6 及更早版本；floating major tag 自動 pick up、pinned SHA / minor / patch 需手動升級，fork PR 若要 checkout 則需明確設 `allow-unsafe-pr-checkout: true`。

[^mcp]: Model Context Protocol，Anthropic 主導的開放協議，讓 LLM client 以標準介面連接工具、資料源與外部服務。2026-07-28 RC（release candidate）版 spec 已於 2026-05-21 locked，Streamable HTTP transport 開始要求 `Mcp-Method` + `Mcp-Name` headers、list / resource read 新增 `ttlMs` + `cacheScope`（模仿 HTTP Cache-Control）、feature lifecycle 明定「Active → Deprecated → Removed」三段、最短 12 個月 deprecation-to-removal 保證。

[^siir]: 服務業創新研發計畫（Service Industry Innovation Research，SIIR），經濟部產業發展署對台灣服務業者的研發補助案，一般案最高 150 萬 / 家、聯合案 500 萬、國際合作 1,000 萬。115 年度第一梯次原本 2025-12-17 至 2026-01-30 收件，但補助經費已於 2026-05-20 用罄，經濟部明文公告不再受理第二梯次，是台灣中小企業補助 stack 中最大一條線的暫時關閉。

[^claude-code]: Claude Code 為 Anthropic 官方 CLI，把 Claude 包成本地 agent，可跑 slash command、hooks、subagent、MCP server 與 workflow orchestration。v2.1.202（2026-07-06 release）新增 `Dynamic workflow size` 進 `/config`（small / medium / large agent counts 為 advisory 而非硬 cap）、`workflow.run_id` + `workflow.name` OTel attributes 進 workflow-spawned agents telemetry——一次 workflow run 全部 agent 活動可從 OTel data 重建，直接對接 Grafana / Honeycomb。
