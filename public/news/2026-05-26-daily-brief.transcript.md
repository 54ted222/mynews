今天想聊三件對獨立開發者來說，這禮拜就會有立即影響的事。不是那種「未來可能怎樣」的趨勢觀察，而是接下來幾天你手上的產品、定價、甚至客戶話術，都得跟著動一動的東西。

先講第一件，最大條的，Google 的 Gemini Spark。Spark 是 Google 在五月十九號的 I/O 大會上揭曉的個人 AI 代理，它的定位很特別，是一個全天候、會主動幫你做事的助理。它常駐在雲端，意思是你把筆電關掉它也還在跑，而且深度串接你的 Gmail、Docs、簡報，還有 Slack 這些工作區的工具。它底層跑的是 Gemini 3.5，加上 Google 自家的 Antigravity 執行框架。不過它也不是完全放飛，碰到那種高風險的動作，比方說要花錢、要替你寄出一封信，它會先問過你才動手。

那為什麼說這禮拜就有影響呢？因為原本 Spark 只開放給少數受信任的測試者，但從五月二十六號這一週開始，它要對美國的 AI Ultra 訂閱用戶開放 beta 了。更關鍵的是價格。Google 在 I/O 上把 AI Ultra 這個方案從原本一個月兩百五十美金，直接砍到一百美金。簡單說，個人代理這個市場的價格錨點，一口氣往下沉了。

這對 indie 有兩層意思。第一，如果你本來想賣「幫人設定個人 agent」這類顧問服務，買方心裡的價格預期已經被重設成一百塊這個級距，你得照著這個新錨點去談。第二，這其實也開了一個縫。Spark 只接 Google 自己的生態，所以那些非工作區的垂直場景，像是 Notion、Linear、或你自家的 SaaS，它管不到。也就是說，「非 Google 生態的主動式代理」這塊，還是獨立開發者的地盤。而且接下來大概三週會是熱期，很適合寫一篇對比的內容，把 Spark、ChatGPT 的代理、還有 Claude 這三家，分別能接哪些 app、資料邊界畫在哪、哪些動作會先問再做，整理成一張決策表，這種結構化的中性比較目前市面上還沒人好好做。

再來是第二件，Anthropic 的募資。彭博在五月二十二號爆料，Anthropic 一輪超過三百億美金的募資，最快這禮拜就要收掉。領投的是 Sequoia、Dragoneer、Altimeter、Greenoaks 這幾家，每家大約出二十億，估值衝破九千億美金。這其實是接著五月二十三號那篇情報的延續，當時提到他們第二季預期有一百零九億的營收，還有史上第一次的季度營業利潤。

這對你的意義在哪？如果你正在跟客戶解釋「為什麼選 Claude 不選別家」，現在可以再加一條很強的論點，就是三百億到帳之後，未來十二個月的營運跑道、品牌、還有通路紅利基本上都確定了。對那些要簽十二個月以上長約的中小企業或大企業客戶來說，這是一顆很有份量的定心丸。反過來講，如果你過去是靠「這些燒錢的前沿實驗室隨時可能漲價、不穩」這種論點，去推銷對手陣營的，這禮拜你的說法會被「Anthropic 首次盈利、又拿了三百億」削掉一塊。

第三件，是 Supabase。它五月的開發者更新這禮拜落地了，重點是一個叫 @supabase/server 的跨執行環境 SDK。白話說，以前你要在 Edge Functions、Vercel、Deno、Bun、還有 Cloudflare Workers 上面各跑一套後端，身分驗證、CORS、用戶端建立這些樣板程式碼得各寫一份。現在這個 SDK 讓你用同一份程式碼跨這五個環境，省掉重複工。除此之外還有幾個東西：Data API 改版，可以針對每一張資料表、每一個函式單獨開關，精準控制哪些東西要暴露出去；Supabase 也進了 Stripe 的市集；還能透過 Wrappers，直接從 Postgres 查 Cloudflare 的 D1 資料庫。

如果你正在跑 Supabase，這禮拜值得花四十五分鐘，挑一條 edge endpoint 試裝看看 @supabase/server，順手把 Data API 的暴露面盤一遍。不過這裡有個時間鎖要提醒，Cloudflare 的 K2.5 要在五月三十號切換到 K2.6，從今天算只剩四天，而且新版比較貴。建議今天就在 staging 環境跑一輪相容性測試，順便估一下帳單會不會跳。

除了這三條主線，還有幾件值得順帶一提。Figma 在五月二十號上線了 Design Agent，可以在畫布裡面用自然語言直接生成、編輯設計，beta 期間免費、不吃 AI 額度，而且同時接了 OpenAI 和 Anthropic 兩家模型。Vercel 的 AI Gateway 也加進了阿里巴巴的 Qwen 3.7 Max，還多了用指令做權重流量切分的新功能。另外法律領域的 Harvey 推出了五百多個現成的用例代理加上 Agent Builder，這代表垂直巨頭開始把「現成代理加上建構器」當成護城河，做單點工具的獨立開發者得往他們沒覆蓋的窄垂直退守，像是繁中合約、台灣或亞洲的法遵這種在地題目，大概還有十二個月的窗口。

重點是，這禮拜你真正要動手的就三件：一，留意 Gemini Spark 五月二十六號開美國 beta、AI Ultra 降到一百塊，重設了個人代理的價格錨點；二，記住 Anthropic 三百億募資本週收掉，把它寫進你的客戶話術；三，趕快試一下 Supabase 的跨環境 SDK，同時盯緊五月三十號 K2.5 切到 K2.6 的倒數，只剩四天。把這三件處理好，這禮拜就不算白過了。
