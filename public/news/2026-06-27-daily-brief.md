[^distillation]: Distillation（模型蒸餾）：用一個強模型（teacher）大量產生問答 / 推理過程，再拿這些輸出訓練較小或自家模型（student），藉此「便宜地」吸收 teacher 的能力。對被蒸餾的 frontier lab 而言常違反 API 條款，因為 student 等於白嫖數十億美元訓練成本得到接近能力的副本。

[^trainium]: Trainium2 / Project Rainier：AWS 自研 AI 訓練晶片 Trainium 第二代，採 systolic array 設計、對標 NVIDIA H100；Project Rainier 是 Amazon 為 Anthropic 客製、跨多個 region 的超級電腦叢集，已部署百萬顆 Trainium2、規劃 Q4 上線 1GW，是 Anthropic 擺脫對 NVIDIA + Azure 依賴的關鍵基建。

[^dodo]: Dodo Payments：印度起家的 Merchant of Record（MoR）服務商，幫 indie SaaS 一次處理全球 VAT / GST / sales tax 申報；4%+40¢ 統一費率、無月費，定位介於 Polar（便宜但需升 Pro 才有競爭力）與 Lemon Squeezy / Paddle（5%+50¢ 但成熟）之間，常被中文社群忽略。
