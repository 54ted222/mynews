- **OpenRouter[^openrouter] 同台揭 Gemini「2-tier router slot」新坐標**：(1) **Gemini 3.5 Flash** $1.50 input / $9 output per M、1M context、4x 速度——定位 **mid-tier 取代 Pro**；(2) **Gemini 3.1 Flash Lite Preview** $0.25 input / $1.50 output per M——是 3.5 Flash 的 **1/6 價**、定位 cheap router；亦即 OpenRouter 5/19 後 Google 系統一條 **3.5 Flash（mid-tier）+ 3.1 Flash Lite（budget）** 的兩段配置可直接接；indie 多模型 router 從原本「Claude Haiku 4.5 / GPT-5.5 Instant / Gemini 3 Flash」三家 budget slot 升級成「Claude Haiku 4.5 / GPT-5.5 Instant / Gemini 3.1 Flash Lite / Gemini 3.5 Flash（mid）」分層；對自家 SaaS 跑「同 prompt × 4–6 家 budget」router 觀察月帳變動本週是熱期
- **Supabase 5/22 token endpoint 切換倒數 1 天 + 一連串時間鎖再啟**：(1) **5/22**（明天）`/v1/oauth/token` 從 HTTP 201 改 HTTP 200 OK——自寫 client 解 status code 的會壞、官方建議改 `response.ok` 範圍判斷；(2) **5/30** 同切換成所有新 project 預設 + Cloudflare K2.5 → K2.6 alias 切換；(3) **6/15** Anthropic Agent SDK Credit Pool 上線；(4) **7/1** Postgres 14 自動升 17；(5) **6/18** Gemini CLI sunset（剩 28 天）；indie 今日重點：跑 Supabase 後端的 staging client 跑一輪 status code parse audit、自家 OAuth client 用 `response.ok` 而非 status code 比對
- **Anthropic Claude Developer Platform 新「3 件套」研究 preview 同步揭**：(1) **Advisor Strategy（beta）**——讓 agent 在主 model 旁諮詢一個 advisor model、跑 high-stakes / 跨領域決策時 intelligence boost；(2) **Web Search / Web Fetch 內建**——agent 直接補上 real-time 知識、不用接外部 search SDK；(3) **Code Execution 內建**——直接在 API call 內跑 Python 分析資料、不用自架 sandbox；對 indie：原本要自己接 OpenAI search + 自架 Python sandbox + 自寫 advisor router 三件事，5/20 起本週可改成「Claude API 一個 endpoint 全包」、自家 wrapper SaaS 中介層摩擦變小、但這也意味「Claude wrapper SaaS」的差異化壓力升高

[^karpathy]: Andrej Karpathy 是 OpenAI 2015 年共同創辦人之一、前 Tesla AI 主管（領 Full Self-Driving 與 Autopilot 計劃）、2024 年離 OpenAI 創辦 Eureka Labs（AI 教育新創）；2026 年 5/19 加入 Anthropic、在 Nick Joseph 麾下帶領新成立的「以 Claude 加速 Claude pre-training」 research team；其 YouTube 上 nanoGPT、micrograd、LLM from scratch 系列影響 5 年以上的獨立工程師 LLM 學習路徑。

[^claudefinance]: Claude Finance 是 Anthropic 2026 年 5/19 Code w/ Claude London 揭的「金融行業 pre-built Managed Agents 套件」，內含 10 個 ready-to-deploy agent：earnings analysis、portfolio review、cap table 管理、due diligence、deal memo、KYC、AML、模型風險、合規檢核、IPO prospectus draft；對打 Salesforce 行業 AI 與摩根 / 高盛自家 AI 平台，主打 5,000+ 人金融機構 ICP。

[^antigravity-sdk]: Antigravity SDK 是 Google 於 2026 年 5/19 I/O Day 1 揭、5/21 進 on demand 階段的 agent 平台開發套件，提供 programmatic 介面控制 Antigravity agent harness，並支援把 agent 部署到開發者自帶的 infra（如 Cloudflare Workers、Vercel、自架 K8s）等第三方 hosting 路徑；對應 Anthropic Self-hosted Sandboxes，是 Google 系統內首個公開揭露的 agent SDK + 自帶 infra hosting 組合。

[^managed-agents-gemini]: Managed Agents in Gemini API 是 Google 於 2026 年 5/19 I/O 揭的 agent-as-a-service 端點：開發者一個 API call 即可取得 fully provisioned agent 與 remote sandbox（不需自架 sandbox 或 orchestrator），底層走 Antigravity harness、預設搭配 Gemini 3.5 Flash；對打 Anthropic Managed Agents「Console + CLI 配置」路線，主打更低的 onboarding 摩擦。

[^openrouter]: OpenRouter 是 2023 年上線的多模型路由平台，把 Anthropic / OpenAI / Google / Mistral / DeepSeek / Meta 等多家 LLM 統一成單一 OpenAI 相容 API endpoint，開發者可用同一支 SDK 切換或同時呼叫不同家模型、即時看到各家 input / output 每百萬 token 的有效報價；常被 indie 拿來做「同 prompt × 多家」budget / mid / frontier 三層路由與月帳比對的基準平台。
