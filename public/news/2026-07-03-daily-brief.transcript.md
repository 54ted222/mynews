今天想聊七月三號這一天，對台灣獨立開發者影響最大的幾條動態，還有幾條進入倒數階段的死線。

先從最有份量的講起。加州州長 Newsom 昨天七月二號正式宣布，加州跟 Anthropic 簽了一個 state-level 的政府採購框架，把 Claude 一次推進加州所有的州政府機關、加上加州境內所有的城市、加上所有郡。這是全美第一個大州直接把某個 frontier lab 的模型定成 preferred vendor。以往政府部門要用 AI，得每個單位自己寫 RFP、自己找廠商、自己談合約，過程碎片化又慢。加州這次是一次覆蓋三層——state agencies、cities、counties——三十天內看其他州會不會跟。對賣美國州級或市級客戶的獨立開發者來說，這是個很直接的 narrative 換擋：以前 pitch Claude 時你講的是「我們選 Anthropic 因為 IPO、因為 talent」；今天起你可以直接說「加州官方 preferred vendor」。就算你在台灣、日本、韓國，這個 signal 也不是無關——因為它給了你一個對本地政府講「你們政府是不是也該有這種 centralized procurement framework」的樣板。順帶一提，這條 signal 跟六月底 Mythos 5 進 Glasswing 兩百家 partner、Fable 5 export ban 全撤、Anthropic 秘密遞 S-1、跟 Amazon 一百億對 5GW Trainium 這四條，其實是「政府 + 五角大廈 + IPO + 敏感 vertical」同一個大 narrative 的四根柱子。

再來講 Claude Sonnet 5。這顆是六月三十號 GA 的，官方數字先出，SWE-bench Verified 八十二點一、OSWorld Verified 八十八點三，比人類 baseline 高十六個百分點。但只有官方數字，客戶會問「你們是不是自己 cherry-pick」。這一週的關鍵補充是——Cursor 官方 blog 揭了他們內部生產環境的 harness 數字。CursorBench 這條，Sonnet 5 拿到六十一點二，對比 Sonnet 4.6 的四十九，一次跳升十二點二個百分點，Cursor 自己說這是「近幾代 Sonnet 相鄰版本」歷來最大單次進步。Terminal-Bench 2.1 這條跳升十三點四、FrontierCode 這條跳升二十三點七。除了 harness，還有 launch partner 的實測——AI for Business 的 GM Yusuke Kaji 說他們讓 Sonnet 5 跑了幾十個「最難的真實 PR」，一次到位 tested and verified。testers 還報告有人看 Sonnet 5 一次連跑「寫一個能重現 bug 的 test → 修 bug → 用 stash 驗證 bug 真的被修掉」多步驟。所以現在你要寫客戶 RFP 佐證，可以直接列四軌同向：Anthropic 官方 SWE-bench、Anthropic 官方 OSWorld、Cursor 生產 harness、還有 launch partner 案例。有這種對照表，比只講一組數字抗打太多。順帶提醒一下 router 這件事——所有 config 裡的 `claude-sonnet-4-6` 這個 model id 該全部換 `claude-sonnet-5`。價格是 promo 期 $2 進 $10 出，到八月三十一號，之後跳回 $3 進 $15 出。新 tokenizer 對同一段文字產出多約三成 tokens，所以雖然單價便宜，實效省下的大概是十二個百分點，不是三分之一。八月三十一號到期前給客戶開個 dashboard 倒數。

第三條是 Microsoft 的動作。GitHub Copilot 這禮拜給 Business 跟 Enterprise 訂閱推了一顆自家的 coding model，叫做 MAI-Code-1-Flash。標榜快、低延遲、適合 high-volume agentic workflow。這件事意義比它的規格重要——這是 GitHub 第一次把自家 in-house coding model 推進 Copilot 的 default routing。過去 Copilot 是被 Anthropic 跟 OpenAI 兩家 host 掉的，這次 MAI-Code 進來，Anthropic Sonnet 5、OpenAI GPT-5.6 Sol、Microsoft MAI-Code 三軌並行的 routing 藍圖正式成型。你賣 enterprise 客戶時可以說「選 Copilot Business = 三軌 model routing + JetBrains 加了 Codex Agent Provider public preview + Copilot Vision 全 tier」，這比 Cursor 或 Claude Code 單 vendor lock-in 好講。JetBrains 那邊也同步 ship 了 Codex 進 Agent Provider public preview，加上 Agent Customizations editor hooks、手動建立 customization file、新的 slash commands。Copilot Vision 七月一號對所有訂閱層打開，包含 Free，沒有 admin toggle。plugin 三頭發行也正式成主流——同一個 plugin 現在該同時 ship 給 Cursor、Codex、Copilot 三家。

第四條是死線倒數。Anthropic 七月八號生效的 KYC——也就是 Free、Pro、Max 消費層要用 Persona 上傳政府 ID 加自拍——今天只剩五天。Team、Enterprise、Platform 訂閱依 Commercial Terms 豁免。這件事有幾個 controversy 值得寫進 case study：Persona 二月才發生過 data exposure，Discord 因此撤掉他們的 age verification vendor；Persona 是 Peter Thiel 的 Founders Fund 投的，Thiel 同時也是 Anthropic 投資人，交叉持股引起質疑；再來，中國國家 ID 卡不被接受，中國用戶沒有護照就等於被鎖。你如果自家 SaaS 有走 consumer Claude login pass-through，尤其含中國、東南亞、北非用戶，二十四小時內做一遍 flow audit——七月八號起 flagged account 被打斷，下游 SaaS 就會 breakage。給客戶 SLO 加兩個 placeholder，一個叫 geo exclusions、一個叫 flagged trigger criteria。

尾巴再快速掃幾條倒數線。Vercel 六月二十五號 ship 了 AI SDK 7，最大的兩件事是 WorkflowAgent 跟第一次官方支援 MCP Apps——這是第一個 indie 可以直接跑的「stateless MCP + durable executor」stack。搭配 MCP 官方七月二十八號 spec RC 倒數二十五天，自家 MCP server 該進入七月十三號前 audit stateful session、七月十四號到二十八號改 stateless + Tasks 的兩段 timeline。GitHub actions/checkout v7 七月十六號 backport 死線倒數十三天，floating major tag 會自動 pick up、pinned SHA 要手動。OpenAI Codex Remote 從 GA 前的 Pro Business Enterprise 三 tier 擴到「所有 ChatGPT 訂閱」都能用，還多了 DigitalOcean Droplet Workspace plugin——一人公司從手機用 QR 配對 Mac 或 Windows host、或直接 provision 一台 DO Droplet 的 stack 正式官方化。Replit Effort-Based Pricing 七月二號全量切換今天進第二天，Reddit 反彈訊號還沒完全收斂，Replit 已經對每個用過的 member 補十美元 credits，但止血效果還在觀察。

所以重點是——今天真正該做的事是四件。第一，客戶 SLO 加一個加州 centralized procurement 的 placeholder，賣 US state 或市級客戶的 deck 更新。第二，Sonnet 5 三 harness 對照表寫成 RFP 佐證範本、router config `claude-sonnet-4-6` 全換 `claude-sonnet-5`、八月三十一號 promo 到期 dashboard 開倒數。第三，KYC 七月八號倒數五天，自家 SaaS 消費 tier flow audit 二十四小時內做完，含中國 東南亞 北非用戶的路徑先備 fallback。第四，router 加 MAI-Code-1-Flash 成 high-volume 那一軌，Copilot Business 客戶三頭 plugin 發行 SOP 更新。剩下的 Vercel AI SDK 7、checkout v7 倒數十三天、Codex Remote 全 tier 這幾條，是明天到下週的中期功課。
