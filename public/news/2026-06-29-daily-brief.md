[^annex-a]: Annex A 指美國商務部長 Lutnick 在 2026-06-26 致 Anthropic CEO Tom Brown 信函中的附錄清單，標題為「Anthropic US Entities – Approved」。被列入 Annex A 的 Anthropic 美國法人主體、其外籍員工，以及美國聯邦民用機關與國家實驗室，使用 Mythos 5 不需另外申請出口許可；Fable 5 仍排除在外。

[^mcp-apps-sep-1865]: MCP Apps 是 Model Context Protocol 2026-07-28 RC 的提案 SEP-1865（Specification Enhancement Proposal 第 1865 號）。允許 MCP server 將互動式 HTML 投送到 host 端的沙箱化 iframe，UI 與 host 共用同一條 JSON-RPC base protocol，所有 UI 觸發的 action 都走 host 既有的 audit / consent 流程，避免 server 自行另開渠道。

[^acp]: ACP（Agent Client Protocol）是由 Cognition 主導的開放協定，目標是讓任意 AI agent 與任意 host（IDE、CLI、桌面 app）之間以標準介面對接。與 MCP 偏 server 端「工具與資料來源」不同，ACP 規範的是 agent runtime 與 client 容器之間的互動，因此 Devin Local 與 Devin Desktop 之外的 host 也能呼叫同一個 agent。

[^devin-local]: Devin Local 是 Cognition 7/1 起在 Devin Desktop 內取代 Cascade 的本地 AI coding agent 版本。以 Rust 重寫、支援 parallel subagents，官方數據比 Cascade 少約 30% token 消耗，並原生支援 ACP。原 Cascade 的 settings、extension、keybindings、MCP connection 會自動 port，但 CI 中顯式呼叫 `cascade` 命令的 script 需手動改為 `devin-local`。

[^stripe-mp]: Stripe Managed Payments 是 Stripe 在 2026 年整併 Lemon Squeezy 後推出的 Merchant of Record（MoR）產品，由 Stripe 代為承擔銷售方角色、處理跨境稅務與發票。2 月起 public preview 已覆蓋 35+ 國，費率 5% + 50¢，與 Lemon Squeezy 對齊；對 indie 而言是 Polar Pro $20/mo 訂閱費之外的另一條無月費 MoR 路線。
