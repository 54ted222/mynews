今天想聊一個對所有 AI 創業者來說節奏會被打亂的日子，就是 5 月 19 號。為什麼？因為今天同一天有兩場重要活動正在進行式，一場是 Google I/O Day 1，台北時間明天凌晨一點開講；另一場是 Code with Claude London，台北時間今晚八點開直播。兩家最大的 AI 玩家在同一天主場開講，這個畫面在 2026 年很少見。

先講 Google 這邊。昨天就有 leak 出來說，今天 keynote 會揭曉四條主軸：Gemini 3.2 Flash、Gemini Omni、Gemini Spark，還有一個全新的 Aluminium OS。其中最有訊號的是 Gemini 3.2 Flash，因為它根本不是「即將上架」，而是 5 月 5 號就已經悄悄出現在 iOS 的 Gemini app 和 AI Studio 後台，有好幾家媒體本週確認真的叫得到、跑得起來。leak 推估的 token 定價是輸入每百萬 token 兩毛五美金、輸出兩塊美金，比目前的 Gemini 3 Flash 便宜大概三成三。換句話說，今天 keynote 比較像是把「既成事實」公開講出來。所以對跑 multi-LLM router 的人來說，今晚 keynote 一結束，最重要的事不是看簡報，是去申請 API key、把這個 Flash 槽位接進你的 router，跑同一批 prompt 跟 Claude Haiku 4.5、GPT-5.5 Instant、Kimi K2.6、DeepSeek V4 Flash 對比，預期月底 router 平均 token 成本可以降一成八到三成。

再來是 Code with Claude London。今天的 livestream 跟錄影全開放，重點是同步加開一場 5 月 20 號的 Extended 場次，主打對象明寫是「獨立開發者跟 early-stage founder」，內容是 managed agents、memory、寫 evaluations、組多 agent 系統、agent battles。這個訊號很清楚，Anthropic 在 SF 場主打 enterprise 之後，這次特別給 indie 留一場 hands-on。

把鏡頭拉開講錢的事。Bloomberg 在 5 月 12 號爆出來，Anthropic 正在談一輪 $30B 募資、估值 $900B，月底前可能就簽 term sheet。如果簽下去，等於超過 OpenAI 三月那輪 $852B 的估值，直接坐上 AI lab 估值第一。對比之下，Anthropic 才在二月那輪 $380B，現在是接近三倍跳。背後撐得起這個估值的，是他們 Q1 2026 annualized 跑到大約 $44B，相當於 80 倍年增；超過一千家客戶每年付 $1M 以上，Fortune 10 裡面有 8 家是穩定客戶。Dario Amodei 5/6 在 SF 揭的。簡單說，這個訊號對 indie 有兩層意義：好的一邊是，未來 12 個月 Claude API 供給結構會很穩，因為估值跟算力都鎖好了；不太好的一邊是，5/13 他們同步發布的 Claude for Small Business 一口氣推了 15 個 agentic workflow，從 QuickBooks、PayPal、HubSpot、Canva、Docusign 通通接好，這代表 Anthropic 正在從 enterprise 往下打 SMB。所以如果你的 SaaS 主打「我是 Claude wrapper 給小公司用」，你的 moat 從這週開始就在縮。

再來一條更有衝擊力的消息。5 月 13 號，一家叫 Recursive Superintelligence 的公司出 stealth，募了 $650M、估值 $4.65B。創辦人陣容很恐怖：Richard Socher，前 Salesforce 首席科學家、You.com 創辦人；Peter Norvig，那本人工智慧課本的作者；Tim Shi，Cresta 共同創辦人；再加上五位從 DeepMind、OpenAI、Meta 出來的研究員，總共八個人。Nvidia、AMD、GV、Greycroft 都進場。他們目標寫得很白：mid-2026 公開「Level 1 autonomous training system」，意思是 AI 自己找自家弱點、自己 redesign 修補、無人介入。對應到 indie，這威脅到五個生態位：fine-tune 中介、RFT 中介、eval harness 工具、prompt management、human-in-the-loop labeling。如果你的 SaaS 在這五條之一，14 個月內要嚴肅評估 pivot。

第三條主線是 Linux Foundation 5 月成立的 Agentic AI Foundation，簡稱 AAIF。OpenAI 跟 Anthropic 共同 co-found，這個畫面本身就值得記一下。AAIF 接收三個大專案：Anthropic 捐 MCP、Block 捐 goose、再加 AGENTS.md。同時 Goose 升到 v1.26，內建 llama.cpp 本地推論，意思是不用 Docker、不用 Ollama、不用 API key，可以完全離線跑。然後把原本的 Computer Controller 重寫叫 Peekaboo，專做 macOS 桌面自動化，採「先看截圖、再點、再打字」這個流程，可以辨識 UI 元素、跑鍵盤、滾動、拖曳、menu、dialog。等於把「本地 LLM 加 macOS GUI 自動化」這件事變成免費、開源、可組合的積木。所以如果你做 macOS 自動化或 Zapier 替代，這週就要把 Goose 加 Peekaboo 列為 baseline。

接下來是微軟。Build 2026 確定改在 6 月 2 號到 3 號、舊金山 Fort Mason 開，剛好落在 Google I/O 後 14 天，等於微軟有兩週可以根據 I/O 動作再調整 keynote。Build 同期 GitHub 推一張新認證叫 GH-600 Agentic AI Developer，這張不是考你寫 code，是考你怎麼 operate、supervise、integrate AI agent 進 production，含 evaluation、rollback、成本監控、MCP 整合。首批 100 名 5/31 前報考有八折。對做 agent ops consulting 或想轉型的 indie，這是門檻很低的個人 signal，建議這個月內考完。

最後一條工具更新，Supabase 5/18 釋出 CLI 2.100.0，把昨天提的三重時間鎖一次補齊：5/18 pg_graphql 預設關、5/22 OAuth 2.1 token、7/1 Postgres 14 升 17。同時推一個跨 runtime 的 Auth SDK，意思是同一份 client code 可以跑 Edge Functions、Vercel、Deno、Bun、Cloudflare Workers，CORS 跟 auth 一致。還有 GitHub secret key 的 Push Protection，commit 前自動掃 Supabase key 防止外洩。跑 Supabase 後端的這週就升上去跑一次 audit。

重點是，給創業者三條今天就能做的 action。第一，今晚八點 Code with Claude London 直播看完之後，把 5/20 那場 Extended 報名好；明天凌晨一點 Google I/O 看完之後，列出「我家 SaaS 在 Gemini Spark、Omni、3.2 Flash、Aluminium OS 各條下的三條曝險」，半小時搞定。第二，如果你跑 Claude 為主，這週寫一份「Anthropic $30B / $900B 估值衝擊下、我家 12 個月 moat」的 audit 草稿，順便看看 Claude for Small Business 那 15 個 workflow 有沒有踩到你的地盤。第三，如果你做 fine-tune、RFT 或 eval harness，這週寫一份「Recursive Superintelligence Level 1 出來之後，我家還剩什麼 moat」的 checklist，順便決定 5/31 前要不要去考 GH-600 拿個八折的個人 signal。今天的節奏就是這樣，雙場直播加三個錢的訊號，調整一下你的這週 calendar。
