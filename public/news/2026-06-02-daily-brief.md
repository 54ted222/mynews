[^polaris]: Project Polaris 為 Microsoft 自家研發的旗艦 coding 模型，採 MoE（Mixture of Experts）架構並依語言／框架切多個 sub-module，於 2026 年 6 月 2 日 Build keynote 揭曉；官方宣稱在 HumanEval、MBPP 等基準勝 GPT-4 Turbo，定位直接對打 Anthropic Claude Code，預計 8 月起取代 GPT-4 Turbo 成為 GitHub Copilot 預設模型、保留三個月 fallback 期。

[^waf]: Windows Agent Framework（WAF）為 Microsoft 開源的多代理人（multi-agent）開發框架，v1.0 於 2026 年 6 月 Build 公布並改採 MIT 授權，同時支援 Python 與 .NET；agent 以 YAML manifest 定義，同一份描述可在開發筆電、Windows 365 Cloud PC 與 Azure 之間 escalate 執行。原始碼位於 github.com/microsoft/agent-framework。

[^chrome-mcp]: Chrome DevTools MCP 為 Google 於 2026 年釋出的 public preview MCP（Model Context Protocol）伺服器，讓 AI coding agent 透過 DevTools 介面直接觀察瀏覽器執行階段行為與 Performance Insights，可被 Cursor、Claude Code、Codex 等 IDE agent 共用，用於 debug 前端執行結果與效能瓶頸。

[^azure-agent-mesh]: Azure Agent Mesh 為 Microsoft 6/2 Build 預覽的跨環境 agent 控制平面，可在本地 Windows server、Windows 365 Cloud PC 與 Azure Arc edge 節點之間 federate 同一份 YAML agent manifest，依延遲與 GPU 可用度自動路由；定價採 consumption-based 並含專屬 agent compute SKU，預計 2026 年 Q4 GA。

[^war]: Windows Agent Runtime（WAR）為 Microsoft 內建於 Windows 的 OS 級 agent 執行環境，以 WinRT 底層 background service 管理 agent lifecycle、記憶與權限；2026 年 6 月以 Insider preview 落地，初期僅支援文字型 agent、輸入限 JSON／XML／PDF，不涵蓋 vision 或通用 tool call。與棒球統計指標 WAR（Wins Above Replacement）無關。

[^composer-2-5]: Cursor Composer 2.5 為 Cursor IDE 於 2026 年 5 月 18 日推出的編碼 agent 模型，主打 sustained 多步 long-running task 穩定性，宣稱以 25 倍合成任務量相對 Composer 2 訓練；定價為輸入 $0.50／百萬 token、輸出 $2.50／百萬 token，另有 fast variant $3／$15，首週推 double usage promo，被視為 GitHub Copilot 漲價後的主要遷移目的地之一。
