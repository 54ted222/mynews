今天想聊 5 月 22 號這天，對一人團隊、小團隊 indie 開發者衝擊最大的三件事。先說結論：第一條是 Supabase 的 OAuth token endpoint 今天正式切換上線，技術細節雖然很小、但 48 小時內要警覺；第二條是 Brett Adcock 第三家新創 Hark 昨天揭了七億美元的 A 輪、估值六十億，把 personal AI 硬體這條賽道的時鐘重新打開；第三條是 Microsoft 前天開源了兩件 agent 安全測試的框架，把這個原本散在各家 OSS 的領域，一口氣集中成 Microsoft 主導的 baseline。我們一條一條看。

先講 Supabase。今天 5 月 22 號是切換日，事情很單純：他們的 `/v1/oauth/token` 這個端點，回應碼從原本的 HTTP 201 改成 HTTP 200 OK。為什麼要改？因為 OAuth 2.1 spec 的第 3.2.3 節明確規定要回 200，回 201 其實是 non-compliant 的，過去也確實造成過一些嚴格 OAuth client 拿 token 失敗的案例。重點是：如果你的程式碼是用 `response.ok` 或是檢查 2XX range 這種主流寫法，今天完全不受影響；但如果你或你抄的範本是寫死 `=== 201` 這種硬比對，那今天起就會炸。所以今天 indie 必做的動作只有兩件：一、跑一輪 staging audit，把自家 OAuth client 看一遍；二、順手把 GitHub repo 上自己散出去的 boilerplate 範本一起改寫。然後要提醒一下，未來 24 天裡其實還有三件事連環疊上來：5 月 30 號 Cloudflare 的 K2.5 要切到 K2.6 別名、6 月 15 號 Anthropic Agent SDK 要開 Credit Pool、7 月 1 號 Postgres 14 會自動升級到 17，所以這 48 小時 horror story 預期會比平常密集，社群討論串值得多盯一下。

再來第二條，Hark。Brett Adcock 這個人你可能不熟，但你大概聽過 Figure.AI 那家做人形機器人的、還有 Archer 那家做電動垂直起降飛機的，這兩家都是他開的，他現在開第三家叫 Hark。昨天 5 月 21 號揭了 A 輪七億美元、估值六十億，2025 年底他自掏一億美元先啟動，現在七十個員工、自己有 B200 GPU 的資料中心，設計主管是前 Apple 的產品高層 Abidur Chowdhury。Hark 定位叫「universal AI interface」，簡單說就是 agentic AI 要能跨桌面、跨手機、跨第三方 app 互相串。今年夏天會先出多模態模型、之後再出 AI-native 的硬體裝置。但真正讓這條新聞變大事的，是他這輪投資人陣容：Parkway 領投，然後 Nvidia、AMD Ventures、Intel Capital、Qualcomm Ventures 四家半導體大廠同框，再加 Salesforce Ventures、Brookfield、ARK Invest、Greycroft、Align Ventures、Prime Movers Lab、Tamarack Global，半導體、雲、SaaS、公開市場基金，六派同框入股一個 A 輪，這個結構過去幾乎沒見過。

那對 indie 來說意義是什麼？personal AI 硬體這條賽道，自從 Humane Pin 失敗、Rabbit R1 評價落地之後，已經沉了大概 18 個月。Hark 是第一個有 Adcock 這個級別的創辦人、加上半導體三巨頭背書的明確 GTM 故事，14 個月內會出貨。所以這條我會分兩面看：機會面是，做「pre-hardware SDK、wrapper、預期生態 plug-in」的本週起有 14 天熱期可以寫內容、發 PoC；威脅面是，如果你家是純軟體的會議筆記、日程管理、個人助理這幾條垂直 SaaS，hardware-native agent 平台一旦落地，12 個月內你的軟體 moat 會被擠到 niche，本週起就該寫一份「為什麼 Hark、Friend、Limitless 上線後我家仍然不同」的客戶 deck。

第三條，Microsoft。5 月 20 號 Microsoft 開源了兩件東西：第一件叫 RAMPART，全名是 Risk Assessment and Measurement Platform for Agentic Red Teaming，是 agent 安全紅隊測試的框架；第二件叫 Clarity，是配對的 observability 與 monitoring 棧。兩件都完整開源，可以接 GitHub Actions、可以接 VS Code 的 testing pane，CI 也跑得起來。RAMPART 最大的設計亮點是 Pytest-native，就是你只要會寫 Pytest，就能寫自家 agent 的紅隊 test case，prompt injection、jailbreak、data exfil、SSRF、tool misuse 這些基線測試他都附了。為什麼這件事很重要？因為過去 agent 紅隊測試這塊是 fragmented 的，PromptFoo、DeepEval、LangSmith、Phoenix、Helicone 各家都有自己的格式跟 GTM，現在 Microsoft 出手把它集中成一個 baseline，本週起這幾家都要面對「為什麼不直接用 RAMPART」這個 reposition 壓力。

對 indie 來說，這條最直接的機會是：跑 agent SaaS 的、做 agent ops 或 safety consulting 的、本週起就把 RAMPART 設成 baseline，寫 5 條自家 vertical 紅隊 case 大概只要半天工作，然後就可以對 SMB 客戶賣「Claude、Antigravity、Codex agent 上線前 1 週的 RAMPART safety audit」這種 productized service，定價大概 $1K 到 $3K 一場。再進階一點，把 12 條 vertical 紅隊 case 包成 PDF 加 GitHub OSS template 賣 $29，金融、醫療、法務、政府、SMB ERP、亞洲多語、繁中欺騙這幾條題目 Microsoft 官方範例都沒涵蓋，indie 切進去的縫剛好夠。

把這三條合起來看，會看到一個更大的格局：5 月這個月，Anthropic、OpenAI、Google 三家 frontier lab 都在揭「agent 跑客戶機房」的路線。OpenAI 5/18 跟 Dell 公告 Codex on-prem，Anthropic 5/19 揭 MCP Tunnels 加四家 self-hosted sandbox（Cloudflare、Daytona、Modal、Vercel），Google 走 Antigravity SDK 自帶 infra hosting。同一個月、三家全揭。對做金融、醫療、政府、SMB ERP 整合的 indie agency 來說，這意思是接下來幾週 vendor selection deck 會變成新熱題、$2K 到 $5K 一場的 PoC retainer 是 baseline。然後同月 Google DeepMind 還用大約一億美元、license 加挖人的方式吃下 Contextual AI 二十多個研究員，包含 RAG 早期共同作者 Douwe Kiela——這手法跟 Microsoft 吃 Inflection、Amazon 吃 Adept、Google 吃 Character.AI 是同一套，避開反壟斷又把人才吸走。做 RAG 中介 SaaS 的 indie，12 個月內 moat 預期會蒸發，本週起就該做自家「12 個月 moat audit」。

重點是：今天 5/22 這天，indie 行事曆上最該排進去的事，依照優先順序是——第一，30 分鐘跑一輪 Supabase OAuth client 的 `response.ok` audit；第二，如果你做 voice agent，花 45 分鐘試裝 Vercel AI SDK 新加的 speech 跟 transcription，跟自架 Whisper、Deepgram 的成本對比一下；第三，如果你做 agent SaaS，花 60 分鐘 fork Microsoft RAMPART 跟 Clarity 到自家 repo，跑 5 條 generic baseline、再補 1 條自家 vertical case。三件事加起來不到三小時，但是這三條今天動跟下週才動，市場聲量差會非常大。今天先這樣，明天再聊。
