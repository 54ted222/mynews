在台灣談線上收款，幾乎逃不開三個名字：**紅陽科技**、**綠界科技** 與 **藍新金流**。三家加起來統治第三方金流市場超過二十年，業內常以「**紅綠藍**」並稱。表面上看，它們都做信用卡、ATM、超商代收、電子發票，費率還都壓在 2.75%–2.8% 區間；但只要你開始算總成本、查申請門檻、實際串 API，就會發現三家根本不是同一種生意。

- **綠界**走的是「**全自助、開發者友善、低門檻**」路線——個人會員月限額 30 萬，文件齊全 5–7 工作天開通，2026 年甚至推出能裝進 Claude Code 與 Copilot CLI 的 AI Skill toolkit，個人副業、indie hacker、新創 MVP 幾乎都從這裡起跳。
- **藍新**定位在「**中小電商主力**」，個人會員 30 天信用卡額度 20 萬、企業 60 萬；分期、定期定額、卡號綁定都做得扎實，但 API 文件需登入才能下載，開發者體驗比綠界悶一截。
- **紅陽**是另一個世界——**沒有真正意義上的個人方案**，年費 12,000、設定費 4,000、徵信費 800 起跳，但信用卡費率可以壓到 1.8%–2%；25 年老牌、線下刷卡機與線上整合並重，適合營業額穩定的實體店家、餐飲、課程業者。

換句話說：**綠界賣的是 self-service、藍新賣的是電商套件、紅陽賣的是議價後的低費率**。這條主軸決定了它們的費率結構、申請文件、撥款週期甚至 API 文件公開到什麼程度——本期五篇就照這條軸來拆。

> 本期所有資料截至 2026-05，引用來源請見各篇文末。費率、額度、撥款週期會隨各家公告變動，正式申請前務必再核對官方頁面。
