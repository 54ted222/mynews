在台灣談線上收款，幾乎逃不開三個名字：**紅陽科技**、**綠界科技[^ecpay]** 與 **藍新金流[^newebpay]**。三家加起來統治第三方金流市場[^ipsp]超過二十年，業內常以「**紅綠藍**」並稱。表面上看，它們都做信用卡、ATM、超商代收、電子發票，費率還都壓在 2.75%–2.8% 區間；但只要你開始算總成本、查申請門檻、實際串 API[^api]，就會發現三家根本不是同一種生意。

- **綠界**走的是「**全自助、開發者友善、低門檻**」路線——個人會員月限額 30 萬，文件齊全 5–7 工作天開通，2026 年甚至推出能裝進 Claude Code 與 Copilot CLI 的 AI Skill toolkit[^aiskill]，個人副業、indie hacker、新創 MVP 幾乎都從這裡起跳。
- **藍新**定位在「**中小電商主力**」，個人會員 30 天信用卡額度 20 萬、企業 60 萬；分期、定期定額、卡號綁定都做得扎實，但 API 文件需登入才能下載，開發者體驗比綠界悶一截。
- **紅陽[^sunpay]**是另一個世界——**沒有真正意義上的個人方案**，年費 12,000、設定費 4,000、徵信費 800 起跳，但信用卡費率可以壓到 1.8%–2%；25 年老牌、線下刷卡機與線上整合並重，適合營業額穩定的實體店家、餐飲、課程業者。

換句話說：**綠界賣的是 self-service、藍新賣的是電商套件、紅陽賣的是議價後的低費率**。這條主軸決定了它們的費率結構、申請文件、撥款週期甚至 API 文件公開到什麼程度——本期五篇就照這條軸來拆。

> 本期所有資料截至 2026-05，引用來源請見各篇文末。費率、額度、撥款週期會隨各家公告變動，正式申請前務必再核對官方頁面。

[^ecpay]: ECPay 綠界科技，股票代號 6763，2022 年 3 月在台灣興櫃市場以承銷價 760 元上櫃，是三家裡唯一掛牌上櫃的金流公司，最大股東是茂為歐買尬集團。長年以「全自助開通、文件公開」的開發者友善路線著稱。
[^newebpay]: NewebPay 藍新金流，股票代號 7739，由智付通與藍新科技在 2018 年合併而來，現為智冠集團子公司（智冠持股約 51%）。產品線重心在訂閱、定期定額與卡號綁定，與遊戲點數卡 MyCard 生態系有歷史淵源。
[^sunpay]: SunPay 紅陽科技，股票代號 7745，2026 年 1 月 5 日才登錄興櫃，大股東是大宇資（持股約 55%）。25 年老牌，產品線從實體刷卡機 EDC POS 起家，後來才把線上金流疊上去，定位偏實體 + 線上整合。
[^ipsp]: 第三方支付（IPSP，Internet Payment Service Provider），指介於買家、賣家與銀行之間的網路支付中介服務商，由商家把收單授權委託給平台，平台再代為向銀行請款並撥款給商家，是電商金流最常見的服務型態。
[^api]: API（Application Programming Interface）指系統之間溝通的程式介面。金流業者把信用卡授權、ATM 轉帳、超商代收等能力包成 HTTP 介面對外開放，讓商家網站可以呼叫並完成收款流程。
[^aiskill]: AI Skill toolkit 指把 API 文件、範例程式碼、錯誤排查經驗整理成適合 AI 程式助理（如 Claude Code、Cursor、Copilot CLI）讀取的知識包，讓開發者用自然語言描述需求，AI 就能直接生成可用的串接程式碼。
